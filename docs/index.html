What is Geometry Dash?

Geometry Dash is a single button skill game.

Runs on:

  • Mobile
  • Desktop
  • Some consoles

Gamemodes

  1. Cube
  2. Ship
  3. Ball
  4. UFO
  5. Wave
  6. Robot
  7. Spider

Controls

Single button press, tapping or holding the button.
In platformer mode, you can also use the arrow keys to move left and right.
However, on mobile they added arrow buttons on the sides of the screen.
